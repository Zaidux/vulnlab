🕵️ Victim Simulator

This page simulates another user browsing the site after a cache-poisoning attack has been planted.

Simulated action: navigating to the cache-poisoned homepage.

In a real attack, the victim would visit the poisoned URL in their browser (with no malicious headers). Nginx would serve them the cached, poisoned response — which loads JavaScript from the attacker's origin.

The attacker-controlled script could then:

  1. Harvest cookies / session tokens
  2. Inject a second-stage XSS payload
  3. Trigger /run?cmd=... for RCE
